# moodee
Mood / activity tracker web app

## Environment
Node: `v16.13.1`
npm: `8.1.2`

## Application
### Run the webserver for the frontend
`node index.js`

### Run the watcher to build the frontend app
`npx babel --watch .\src\ --out-dir .\build\ --presets react-app/prod`

### Open the webpage
http://localhost:3000
